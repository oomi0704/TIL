# Day 01 メモ（AWS SAA）

- **Amazon API Gateway**  
  API を作成・公開・保護・監視するためのマネージドサービス。

- **Amazon Cognito ユーザープールオーソライザー**  
  API Gateway と連携し、Cognito ユーザープールで認証されたユーザーのみ API へアクセスさせる仕組み。

- **Site-to-Site VPN**  
  オンプレミス環境と AWS（VPC）をインターネット経由で暗号化接続する方式。

- **SAML 対応 IdP（Identity Provider）**  
  オンプレミスで運用する場合は、  
  **自社データセンターやサーバー室に設置・運用する、SAML 2.0 準拠の認証基盤システム** を指す。

- **OAI（Origin Access Identity）**  
  CloudFront から S3 オリジンへのアクセスを制御する仕組み。

- **LCR / NSFR**  
  金融機関の流動性規制に関する指標。

- **プロシージャキャッシュ**  
  ストアドプロシージャなどの実行計画を再利用し、処理効率を高めるためのキャッシュ。
